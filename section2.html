the basic language of the web: html

all modern websites and web apps are built using three fundamental technologies: HTML, CSS, JavaScript. these are the languages of the web

in this post, let's focus on html. we will learn what html is all about, and why you too should learn it

what is html?

html stands for hypertext markup language. it's a markup language that web devs use to structure and descript the content of a webpage (not a programming language)

html consits of elements taht describe diff types of content: paragraphs, links, headings, images, video, etc. web browsers understand html and render html code as websites

in html, each elemnt is made up of 3 parts:

  1. th eopening tag
  2. the closing tag
  3. the actual element

you can learn more at mdn web docs.

why should you learn html

there are countless reasons for learning the fundamental language of the web. here are 5 of them:

  • to be able to use fundamental web dev language
  • to handcraft websites instead of using wordpres
  • to build web applications
  • to impress friends
  • to have fun
